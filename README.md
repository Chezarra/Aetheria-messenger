ОПИСАНИЕ: 

Это простой P2P-мессенджер, который работает без серверов.
Сообщения отправляются напрямую между пользователями через браузер.
Работает пока только на ПК.
Вообще я делала это для себя и подруги,но если какая-то заблудшая душа увидит это то...пускай :)

КАК УСТАНОВИТЬ? 

1. Скачиваете файлы
2. Разархивируйте их в любое место
3. Запустите файл index.html через ПКМ - ОТКРЫТЬ С ПОМОЩЬЮ - ВАШ БРАУЗЕР

КАК ПОЛЬЗОВАТЬСЯ? 

1.Пользователь А нажимает «Создать комнату» → появится длинный код в окошке Offer.

2.Пользователь А копирует этот код и отправляет пользователю Б например,в другом мессенджере.

3.Пользователь Б вставляет код в поле Offer, нажимает «Подключиться» → появится код ответа (Answer).

4.Пользователь Б отправляет код Answer обратно пользователю А.

5.Пользователь А вставляет код Answer в соответствующее поле → соединение установлено.

6.Можно писать сообщения!

ВАЖНЫЕ ЗАМЕЧАНИЯ:

1. Мессенджер не сохраняет историю сообщений после закрытия вкладки.
2. Сообщения отправляются напрямую - интернет нужен для соединения между пользователями.
3. Работает на современных браузерах (Chrome, Edge, Firefox).
4. При использовании на разных сетях - соединение может зависеть от NAT/роутера, но стандартный Google STUN уже встроен.
